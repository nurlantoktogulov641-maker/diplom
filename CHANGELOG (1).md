Пользователи (аутентификация, роли, профили)

Маршруты (CRUD, поиск, фильтрация)

Отклики (создание, смена статусов)

Отзывы и рейтинг (оценка 1–5, средний рейтинг)

18.04.2026 – Подготовительный этап. Проектирование
Выполненные задачи:

Разработаны UML-диаграммы для подсистем «Отклики», «Чат» и «Отзывы».

Спроектирована схема базы данных PostgreSQL:

Создана таблица tags (теги маршрутов)

Создана связующая таблица routes_tags (многие-ко-многим)

Создана таблица reviews (отзывы с рейтингом 1–5)

Обновлена ER-диаграмма проекта.

19.04.2026 – Экспериментальный этап. Пользовательский функционал
Выполненные задачи:

Реализована сессионная аутентификация Django (исправлены ошибки входа/выхода).

Настроено хеширование паролей PBKDF2 (стандартный механизм Django).

Включена и настроена CSRF-защита для всех форм.

Реализована ролевая модель:

Гость – только просмотр маршрутов

Пользователь – создание маршрутов, откликов, чатов, отзывов

Администратор – модерация контента и пользователей

Организовано разграничение доступа:

Редактирование/удаление маршрута – только автор

Просмотр откликов на маршрут – только автор маршрута

Чат – только автор и принятые участники

20.04 – 22.04.2026 – Управление маршрутами
Выполненные задачи:

Реализован полный CRUD маршрутов:

Создание – форма со всеми полями (страны, города, даты поездки, бюджет, описание, теги, изображение)

Редактирование – доступно только автору маршрута

Удаление – доступно автору и администратору

Подключена библиотека django-taggit для работы с тегами.

Настроена загрузка изображений через Pillow (ImageField).

Проведено тестирование производительности:

При 1000 маршрутов время отклика страницы списка составило ≤480 мс (с пагинацией 6 маршрутов на страницу).

23.04 – 25.04.2026 – Список и карточка маршрута
Выполненные задачи:

Реализован список маршрутов с:

Пагинацией – 6 маршрутов на страницу

Сортировкой – по новизне (дата создания), бюджету (возрастание/убывание), дате начала

Фильтрацией – поиск по названию (без учёта регистра), фильтр по бюджету (от/до)

Отображение автора (ссылка на публичный профиль)

Отображение тегов, дат, бюджета, описания, изображения

Счётчик просмотров (поле views, инкремент при каждом открытии)

26.04 – 27.04.2026 – Подсистема откликов
Выполненные задачи:

Создана модель Response с полями:

user (кто откликается)

route (на какой маршрут)

status (ожидание / принят / отклонён)

created_at (дата и время)

comment (сопроводительный текст)

Реализована кнопка «Откликнуться» на карточке маршрута:

Доступна только авторизованным пользователям (кроме автора маршрута)

Реализована страница управления откликами для автора маршрута:

Адрес: /routes/<id>/responses/

Отображение всех откликов с возможностью смены статуса (принять/отклонить)

Добавлены flash-уведомления при изменении статуса отклика.

28.04.2026 – Подсистема чата
Выполненные задачи:

Создана модель Message с полями:

room_id (идентификатор комнаты, связка "маршрут – пользователь")

sender (отправитель)

content (текст сообщения)

timestamp (время отправки)

Реализована комната чата по адресу: /chat/<route_id>/

Настроены права доступа к чату:

Автор маршрута

Пользователи с откликом в статусе «принят»

Реализована отправка


сообщений через POST-форму.

Реализовано отображение истории сообщений (сортировка по возрастанию времени).

29.04 – 30.04.2026 – Подсистема отзывов и рейтинга
Выполненные задачи:

Создана модель Review с полями:

author (кто оставляет отзыв)

target_user (на кого отзыв)

route (по какому маршруту)

rating (целое число от 1 до 5)

comment (текст отзыва)

created_at, updated_at (даты создания и редактирования)

Реализована возможность оставить отзыв о попутчике:

Условие – маршрут должен быть завершён (текущая дата > даты окончания)

Реализованы редактирование и удаление отзыва (только для автора отзыва).

Реализован автоматический расчёт среднего рейтинга пользователя:

Добавлено поле average_rating в модель пользователя

Пересчёт происходит при сохранении/удалении отзыва

Реализована страница публичного профиля пользователя:

Адрес: /user/<username>/

Отображается аватар, информация о себе, интересы, средний рейтинг

Выводится список отзывов, оставленных о пользователе
